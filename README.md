# Raylib-fun

## Quick Start for macOS

**Clone**     
`git clone https://github.com/bjwdenbesten/raylib-fun.git`   

**[Raylib:](https://www.raylib.com/)**   
`brew install raylib`   

**Linking - pkg-config:**    
`brew install pkgconf`  

**Compile**     
`make {filename}`
